Tipos de Datos en JavaScript

¿Cuáles son algunos tipos de datos JS?

Volver al índice

Tipos Primitivos

JavaScript tiene 7 tipos de datos primitivos:

String

Secuencia de caracteres utilizada para representar texto.

const nombre = "María";
const mensaje = 'Hola, mundo!';
const template = `El nombre es ${nombre}`;  // Template literals (ES6)

Number

Valores numéricos, incluyendo enteros y decimales. Solo hay un tipo numérico en JS.

const entero = 42;
const decimal = 3.14;
const exponencial = 2.5e6;  // 2,500,000
const binario = 0b1010;     // 10 en decimal
const octal = 0o744;        // 484 en decimal
const hexadecimal = 0xFF;   // 255 en decimal
const infinito = Infinity;
const noEsNumero = NaN;     // Not a Number

Boolean

Valor lógico que representa verdadero o falso.

const verdadero = true;
const falso = false;
const comparacion = 5 > 3;  // true

Undefined

Valor que tiene una variable a la que no se le ha asignado un valor.

let sinDefinir;
console.log(sinDefinir);    // undefined

Null

Valor que representa la ausencia intencional de cualquier valor u objeto.

const valorNulo = null;
console.log(valorNulo);     // null

Symbol (ES6)

Tipo de datos cuyos valores son únicos e inmutables, útil como identificadores de propiedades de objetos.

const simbolo = Symbol('descripcion');
const otroSimbolo = Symbol('descripcion');
console.log(simbolo === otroSimbolo);  // false

BigInt (ES2020)

Permite representar números enteros de precisión arbitraria.

const numeroGrande = 9007199254740991n;
const otroBigInt = BigInt("9007199254740991");
const bigIntLiteral = 1234567890123456789012345678901234567890n;

Tipos de Referencia (Objetos)

Además de los tipos primitivos, JavaScript tiene tipos de referencia, que son objetos y subtipos de objetos:

Object

Colección de pares clave-valor y el bloque de construcción más básico de JavaScript.

const persona = {
  nombre: "Juan",
  edad: 30,
  saludar: function() {
    return `Hola, soy ${this.nombre}`;
  }
};

Array

Objetos especiales utilizados para almacenar secuencias de valores ordenados.

const numeros = [1, 2, 3, 4, 5];
const mixto = [1, "dos", true, null, {clave: "valor"}];
const matriz = [[1, 2], [3, 4]];

console.log(numeros[0]);      // 1
console.log(numeros.length);  // 5

Function

Las funciones en JavaScript son objetos de primera clase que pueden ser pasados, retornados y asignados.

// Declaración de función
function sumar(a, b) {
  return a + b;
}

// Expresión de función
const restar = function(a, b) {
  return a - b;
};

// Arrow function (ES6)
const multiplicar = (a, b) => a * b;

Date

Objeto para trabajar con fechas y horas.

const hoy = new Date();
const navidad = new Date(2024, 11, 25);  // 25 Dic 2024

RegExp

Objeto para trabajar con expresiones regulares.

const patron = /^[a-z]+$/i;
const otraForma = new RegExp('^[a-z]+$', 'i');

const esLetra = patron.test('AbCd');  // true

Map, Set, WeakMap, WeakSet (ES6)

Estructuras de datos especializadas introducidas en ES6.

// Map: colección de pares clave-valor donde las claves pueden ser de cualquier tipo
const mapa = new Map();
mapa.set('clave', 'valor');
mapa.set(42, 'número');
mapa.set({}, 'objeto');

// Set: colección de valores únicos
const conjunto = new Set([1, 2, 3, 2, 1]);  // {1, 2, 3}

Verificación de Tipos

Operador typeof

El operador typeof devuelve una cadena que indica el tipo de un operando.

typeof "Hola";      // "string"
typeof 42;          // "number"
typeof true;        // "boolean"
typeof undefined;   // "undefined"
typeof Symbol();    // "symbol"
typeof 42n;         // "bigint"

// Peculiaridades
typeof null;        // "object" (esto es un error histórico)
typeof [];          // "object"
typeof new Date();  // "object"
typeof function(){}; // "function"

instanceof y Array.isArray()

Para verificar si un objeto es una instancia de una clase específica o si un valor es un array.

const arr = [1, 2, 3];
const obj = {a: 1};
const date = new Date();

arr instanceof Array;    // true
obj instanceof Object;   // true
date instanceof Date;    // true

Array.isArray(arr);      // true
Array.isArray(obj);      // false

Conversión entre Tipos

Conversiones Comunes

// String a Number
Number("42");          // 42
parseInt("42px", 10);  // 42 (base 10)
parseFloat("3.14");    // 3.14
+"42";                 // 42 (operador unario +)

// Number a String
String(42);            // "42"
(42).toString();       // "42"
42 + "";               // "42" (concatenación implícita)

// A Boolean
Boolean(0);            // false
Boolean("");           // false
Boolean(null);         // false
Boolean(undefined);    // false
Boolean(NaN);          // false
Boolean(1);            // true
Boolean("hola");       // true
Boolean({});           // true
Boolean([]);           // true

// Boolean a Number
Number(true);          // 1
Number(false);         // 0

// Object a String
const obj = {nombre: "Juan"};
String(obj);           // "[object Object]"
JSON.stringify(obj);   // '{"nombre":"Juan"}'

Resumen y Buenas Prácticas

JavaScript es un lenguaje de tipado dinámico, lo que significa que las variables pueden cambiar de tipo durante la ejecución del programa.

Consejos para trabajar con tipos en JavaScript:

  • Utilizar comparaciones estrictas (=== y !==) para evitar conversiones de tipo inesperadas
  • Ser cuidadoso con las conversiones implícitas en operaciones matemáticas
  • Considerar el uso de TypeScript para proyectos grandes donde el tipado estático es beneficioso
  • Verificar tipos antes de realizar operaciones que dependen del tipo de datos
  • Manejar adecuadamente los valores null y undefined
  • Utilizar operadores de coalescencia nula (??) y encadenamiento opcional (?.) en ES2020
