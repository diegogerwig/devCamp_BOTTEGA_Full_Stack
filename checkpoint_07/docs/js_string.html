Introducción a los Strings en JavaScript

Los strings (cadenas de texto) son uno de los tipos de datos fundamentales en JavaScript. Representan texto y se crean encerrando una secuencia de caracteres entre comillas simples, dobles o backticks (acentos graves).

Creación de Strings

// Tres formas de definir strings
const str1 = 'Comillas simples';
const str2 = "Comillas dobles";
const str3 = `Backticks (template literals)`;

// Los template literals permiten interpolación
const nombre = "María";
const saludo = `Hola, ${nombre}!`;  // "Hola, María!"

// Caracteres de escape
const lineaNueva = "Primera línea\nSegunda línea";
const tabulador = "Texto\tcon\ttabulaciones";
const comillaEscapada = "Ella dijo: \"Hola\"";

Las Tres Funciones Principales de Manipulación de Strings

Aunque los strings en JavaScript tienen muchos métodos útiles, tres de los más fundamentales y ampliamente utilizados son:

1. substring() / slice()

Estas funciones permiten extraer una parte de un string basándose en índices.

Sintaxis:

string.substring(indiceInicio, indiceFin);
string.slice(indiceInicio, indiceFin);

Ejemplos:

const texto = "JavaScript es increíble";

// Obtener "JavaScript"
console.log(texto.substring(0, 10));  // "JavaScript"
console.log(texto.slice(0, 10));      // "JavaScript"

// Obtener "increíble"
console.log(texto.substring(14));     // "increíble" (hasta el final)
console.log(texto.slice(14));         // "increíble" (hasta el final)

// Diferencias entre substring y slice con índices negativos
console.log(texto.substring(-5));     // "JavaScript es increíble" (convierte -5 a 0)
console.log(texto.slice(-9));         // "increíble" (cuenta desde el final)

Casos de uso:

  • Extraer partes específicas de una cadena
  • Cortar texto para mostrar previsualizaciones
  • Separar texto en diferentes componentes
  • Obtener extensiones de archivos o dominios de URLs

2. replace() / replaceAll()

Estas funciones permiten reemplazar texto dentro de un string con otro texto o patrón.

Sintaxis:

string.replace(buscar, reemplazar);       // Reemplaza solo la primera coincidencia
string.replaceAll(buscar, reemplazar);    // Reemplaza todas las coincidencias (ES2021)

Ejemplos:

const texto = "El perro corre con otro perro";

// Reemplazar la primera coincidencia
console.log(texto.replace("perro", "gato"));  
// "El gato corre con otro perro"

// Reemplazar todas las coincidencias (ES2021)
console.log(texto.replaceAll("perro", "gato"));  
// "El gato corre con otro gato"

// Usando expresiones regulares
console.log(texto.replace(/perro/g, "gato"));  
// "El gato corre con otro gato"

// Usar una función para reemplazar
console.log(texto.replace(/perro/g, match => match.toUpperCase()));  
// "El PERRO corre con otro PERRO"

Casos de uso:

  • Corregir textos o formatos
  • Formatear datos para mostrar
  • Sanitizar entradas de usuario
  • Reemplazar marcadores de posición con datos reales
  • Transformar formatos de texto

3. split()

Esta función divide un string en un array de substrings basándose en un separador.

Sintaxis:

string.split(separador, limite);       // límite es opcional

Ejemplos:

const texto = "manzana,pera,uva,naranja";

// Dividir por comas
const frutas = texto.split(",");
console.log(frutas);  // ["manzana", "pera", "uva", "naranja"]

// Dividir por espacio
const frase = "JavaScript es un lenguaje de programación";
const palabras = frase.split(" ");
console.log(palabras);  
// ["JavaScript", "es", "un", "lenguaje", "de", "programación"]

// Limitar el número de elementos
console.log(frase.split(" ", 3));  // ["JavaScript", "es", "un"]

// Dividir en caracteres individuales
console.log("Hola".split(""));  // ["H", "o", "l", "a"]

// String vacío si no hay coincidencias
console.log("Sin separador".split("-"));  // ["Sin separador"]

Casos de uso:

  • Parsear datos CSV o separados por delimitadores
  • Separar frases en palabras
  • Dividir nombres completos en nombre y apellido
  • Transformar strings en arrays para facilitar su manipulación
  • Tokenizar texto para análisis

Otras Funciones Útiles de String

toUpperCase() / toLowerCase()

Convierten un string a mayúsculas o minúsculas.

"Hola".toUpperCase();  // "HOLA"
"Mundo".toLowerCase();  // "mundo"

trim() / trimStart() / trimEnd()

Eliminan espacios en blanco del principio y/o final.

"  texto  ".trim();      // "texto"
"  texto  ".trimStart();  // "texto  "
"  texto  ".trimEnd();    // "  texto"

indexOf() / lastIndexOf()

Buscan la posición de un substring.

"JavaScript".indexOf("Script");    // 4
"abcdabc".lastIndexOf("abc");      // 4

includes() / startsWith() / endsWith()

Comprueban si un string contiene, comienza o termina con un substring.

"JavaScript".includes("Script");    // true
"JavaScript".startsWith("Java");   // true
"JavaScript".endsWith("ipt");      // true

concat()

Concatena dos o más strings.

"Hola ".concat("Mundo");  // "Hola Mundo"
// Mejor usar operador +:
"Hola " + "Mundo";       // "Hola Mundo"

padStart() / padEnd()

Rellenan un string hasta una longitud específica.

"5".padStart(2, "0");  // "05"
"Hola".padEnd(10, ".");  // "Hola......"

Ejemplo Práctico

Este ejemplo combina las tres funciones principales para procesar y formatear un texto:

// Función que procesa un nombre completo
function procesarNombre(nombreCompleto) {
  // 1. Usar split para separar nombre y apellido
  const partes = nombreCompleto.split(" ");
  
  // 2. Usar slice para extraer el primer nombre
  const primerNombre = partes[0];
  
  // 3. Usar replace para formatear el apellido en mayúsculas
  const apellido = partes[1].replace(
    partes[1], 
    partes[1].toUpperCase()
  );
  
  return `${primerNombre} ${apellido}`;
}

// Ejemplos
console.log(procesarNombre("john doe"));  // "john DOE"
console.log(procesarNombre("ana smith"));  // "ana SMITH"

Otro ejemplo combinando métodos de string para formatear URLs:

// Función que convierte un título en una URL amigable (slug)
function crearSlug(titulo) {
  return titulo
    .toLowerCase()                      // Convertir a minúsculas
    .replace(/[áéíóúüñ]/g, match => {  // Reemplazar acentos
      const reemplazos = {
        'á': 'a', 'é': 'e', 'í': 'i', 'ó': 'o', 'ú': 'u',
        'ü': 'u', 'ñ': 'n'
      };
      return reemplazos[match];
    })
    .replace(/[^\w\s-]/g, '')          // Eliminar caracteres especiales
    .replace(/\s+/g, '-')              // Reemplazar espacios por guiones
    .replace(/-+/g, '-')               // Eliminar guiones duplicados
    .trim();                           // Eliminar espacios iniciales/finales
}

// Ejemplos
console.log(crearSlug("JavaScript: Guía Rápida"));  
// "javascript-guia-rapida"

console.log(crearSlug("¿Cómo funciona String.split()?"));  
// "como-funciona-stringsplit"
