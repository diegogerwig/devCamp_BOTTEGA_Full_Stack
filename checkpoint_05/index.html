¿Qué es un condicional?

Un condicional es una estructura de control que permite ejecutar bloques de código basados en condiciones. En Python, usamos if, elif, y else para crear condicionales. Más información en la documentación oficial de Python.

Sintaxis

if condition:
    # código a ejecutar si la condición es verdadera
elif another_condition:
    # código a ejecutar si la otra condición es verdadera
else:
    # código a ejecutar si ninguna condición es verdadera

Ejemplo

age = 18
if age >= 18:
    print("You are an adult.")
elif age >= 13:
    print("You are a teenager.")
else:
    print("You are a child.")

En este ejemplo, el programa imprime "You are an adult." porque age es 18.

¿Cuáles son los diferentes tipos de bucles en Python? ¿Por qué son útiles?

Los bucles en Python permiten ejecutar un bloque de código repetidamente. Hay dos tipos principales de bucles: for y while. Más sobre bucles for y más sobre bucles while en la documentación oficial.

Bucle for

El bucle for se usa para iterar sobre una secuencia (lista, tupla, diccionario, conjunto o cadena).

Sintaxis

for variable in sequence:
    # código a ejecutar en cada iteración

Ejemplo

for i in range(5):
    print(i)  # Imprime números del 0 al 4

Bucle while

El bucle while repite un bloque de código mientras una condición sea verdadera.

Sintaxis

while condition:
    # código a ejecutar mientras la condición sea verdadera

Ejemplo

count = 0
while count < 5:
    print(count)
    count += 1  # Incrementa count en 1 en cada iteración

¿Qué es una lista por comprensión en Python?

Una lista por comprensión es una forma concisa de crear listas. Permite generar listas aplicando una expresión a cada elemento de una secuencia. Más sobre comprensión de listas en la documentación oficial.

Sintaxis

[expression for item in sequence if condition]

Ejemplo

numbers = [x for x in range(10)]
# Crea una lista de números del 0 al 9

También se pueden aplicar condiciones:

even_numbers = [x for x in range(10) if x % 2 == 0]
# Crea una lista de números pares del 0 al 9

¿Qué es un argumento en Python?

Un argumento es un valor que se pasa a una función cuando se llama. Las funciones pueden aceptar uno o más argumentos para realizar operaciones. Más sobre funciones y argumentos en la documentación oficial.

Sintaxis

def function_name(arg1, arg2, ...):
    # código que usa los argumentos

Ejemplo

def greet(name):
    print(f"Hello, {name}!")

greet("Alice")  # Imprime "Hello, Alice!"

¿Qué es una función Lambda en Python?

Una función lambda es una pequeña función anónima definida con la palabra clave lambda. Puede tener cualquier número de argumentos pero solo una expresión. Más sobre funciones lambda en la documentación oficial.

Sintaxis

lambda arguments: expression

Ejemplo

sum = lambda a, b: a + b
print(sum(2, 3))  # Salida: 5

¿Qué es un paquete pip?

pip es el sistema de gestión de paquetes para Python. Se utiliza para instalar y gestionar bibliotecas y dependencias adicionales que no están incluidas en la biblioteca estándar de Python. Documentación oficial de pip.

Instalación

pip install nombre_paquete

Ejemplo

pip install requests  # Instala la biblioteca requests
